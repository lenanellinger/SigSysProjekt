Idee¶

Mit der normalen Fourier Transformation eines Signals erhlt man Informationen ber den durchschnittlichen Frequenzgehalt des Signals ber die gesamte Dauer. Es sagt jedoch nichts ber den zeitlichen Verlauf der Frequenzen aus.

Durch die Short-Time Fourier Transformation (Teil der Fourier-Analyse) ist es mglich Aussagen ber den Zeitpunkt der Frequenzen zu treffen. Das Ergebnis der STFT stellt die zeitliche nderung des Frequenzspektrums dar. Hier wird nicht das gesamte Signal transformiert, sondern immer nur ein gewisser Ausschnitt des Signals (sogenanntes Fenster). Dieses Fenster wird ber das gesamte Signal Stck fr Stck geschoben und jeweils transformiert mitttels der Fourier Transformation. So erhlt man durch die Fourier Transformation die Spektralbereiche fr verschiedene Abschnitte des Signals. Die STFT stellt nun die Aneinanderreihung der Spektralbereiche im zeitlichen Verlauf dar.

Beispielsignal für die folgenden Erklärungen¶

Wir verwenden fr die kommenden Erklrungen ein Sinus-Signal, welches von 1 Hz auf 5 Hz in Sekunde 5 springt. An diesem Signal kann man einen Frequenzspruch beobachten und somit ist der zeitliche Verlauf des Frequenzspektrums gut untersuchbar.

Wichtige Größen¶

Zwei Gren sind fr die Durchfhrung der STFT entscheidend:

  • Die Gre des Fenster (window_size) in Sekunden
  • Die Schrittweite zwischen zwei Fenstern (hop_size) in Sekunden

In jedem Schritt fhrt man die Fourier Transformation in dem Fenster mit der Fenstergre window_size durch. Dieses Fenster wird fr den nchsten Schritt um die hop_size nach rechts verschoben. Wenn die hop_size kleiner als die window_size ist, so berlappen sich die Fenster.

hop_size = 0.06
window_size = 1

def fourier_transformation_of_window(time, signal, fs, window_pos, window_size):
    """
    Fhrt die Fourier Transformation fr ein bestimmtes Fenster des Signals aus
    und gibt die jeweiligen Plot aus.
    
    Parameter:
        - time: Array der Zeit
        - signal: Array des Signals
        - fs: Abtastfrequenz
        - window_pos: Position des Fenster in Sekunden
        - window_size: Gre des Fensters in Sekunden
    """
    window_size = window_size * fs
    N = len(signal)
    window_pos_frames = int(fs * window_pos)
    window_function = np.zeros(N)
    # setzt die Stellen des Signals auf 1, die im Fenster liegen
    window_function[window_pos_frames : window_pos_frames + window_size] = 1
    # Durch die Multiplikation wird das Signal nur im Fenster ausgeschnitten
    signal = signal * window_function   
    
    # Plotten
    plt.figure(figsize=(10, 3))
    
    # windowed signal
    plt.subplot(1, 2, 1)
    plt.title("Windowed Signal")
    plt.plot(time, signal, c='k') # signal
    plt.plot(time, window_function, c='r') # window function in rot
    plt.xlim([min(time), max(time)])
    plt.ylim([-1.5, 1.5])
    plt.xlabel('Zeit [sec]')
    
    # Fourier Transformation von diesem Window
    plt.subplot(1, 2, 2)
    plt.title("Fourier Transformation of Window")
    
    X = np.abs(np.fft.fft(signal)) / fs # FT durchfhren
    X = X[:N//2]
    freq = np.fft.fftfreq(N, d=1/fs)
    freq = freq[:N//2]
    
    plt.plot(freq, X, c='k')
    plt.xlim([0, 10])
    plt.ylim([0, 3])
    plt.xlabel('Frequenz [Hz]')
    plt.tight_layout()
    plt.show()

Implementierung der Short-Time Fourier Transformation¶

Fr die Short-Time Fourier Transformation wird fr jedes Fenster (der Gre windowsize) die Fourier Transformation durchgefhrt und in einem komplexen Array gespeichert. Durch die anschlieende Quatrierung des Betrags wird der Array real und kann gezeichnet werden. Das Ergebnis wird in einem Spektogramm dargestellt. In dieser Implementation wird die Zeitdiskrete Short Time Fourier Transformation durchgefhrt. Sie wird durch folgende Formel realisiert: $$X(m,w) = \sum{n=-\infty}^\infty x[n] w[n-m] e^{-j\omega n}$$ mit dem Signal x[n], der Fensterfunktion w[n], Kreisfrequenz $\omega$ und dem Analysezeitpunkt m.

def stft(signal, window_size, hop_size, only_positive_frequencies=False):
    """
    Berechnet die Short-Time_fourier Transformation fr ein gegebenes Signal

    Parameter:
        - signal: Signal auf dem STFT durchgefhrt werden soll
        - window_size: Gre des Fensters in Sekunden
        - hop_size: Bestimmt den Abstand zwischen zwei Fensteranfngen in Sekunden
        - only_positive_frequencies: wenn auf True, dann wird nur der positive Part des Spektrums benutzt
    """
    window_size = int(window_size * fs)  # in Frames
    hop_size = int(hop_size * fs)
    N = len(signal)
    window_function = np.ones(window_size)
    # Wie viele Fenster bentigt werden, um das gesamte Signal abzudecken
    amount_of_windows = np.floor((N - window_size) / hop_size).astype(int) + 1
    # Das Ergebnis wird initialisiert. Die Gre wird dabei von der Gre eines Fensters und der Anzahl der Fenster ber das gesamte Signal bestimmt
    # Komplex ist der Datentyp, da das Ergbnis der FFT einen komplexen Array zurckgibt
    stft_signal = np.zeros((window_size, amount_of_windows), dtype='complex')
    
    # Fr jedes Fenster wird die Fourier Transformation durchgefhrt
    for count in range(amount_of_windows):
        # Das Fenster wird innerhalb des Signals ausgeschnitten
        windowed_signal = signal[count * hop_size : count * hop_size + window_size] * window_function
        # Die Fourier transformation wird durchgefhrt in diesem Fenster. Das Ergebnis ist komplex
        stft_window = np.fft.fft(windowed_signal)
        # Das Ergbnis wird nun in einer Spalte des Arrays gespeichert
        stft_signal[:, count] = stft_window
    
    # filtert positive Frequenzen
    if only_positive_frequencies:
        K = 1 + window_size // 2
        stft_signal = stft_signal[0:K, :]
    
    # Das spectrogram ist eine zwei-dimensionale Representation der quatrierten Magnitude der STFT:
    Y = np.abs(stft_signal) ** 2
    
    T_coef = np.arange(stft_signal.shape[1]) * hop_size / fs
    F_coef = np.arange(stft_signal.shape[0]) * fs / window_size

    left = min(T_coef)
    right = max(T_coef) + window_size / fs
    lower = min(F_coef)
    upper = max(F_coef)

    plt.imshow(Y, origin='lower', aspect='auto', 
               extent=[left, right, lower, upper])
    plt.ylim(0, 10)
    plt.xlabel('Zeit [sec]')
    plt.ylabel('Frequenz [Hz]')
    plt.tight_layout()

Wie wähle ich die Fenstergröße? Küpfmüllersche Unbestimmtheitsrelation¶

Eine Eigenschaft der STFT ist die Kpfmllersche Unbestimmtheitsrelation: Sie beschreibt den Zusammenhang zwischen der Auflsung im Zeitbereich und der Auflsung im Frequenzbereich. das Produkt aus Zeit und Frequenz ist stets konstant.

  • Mchte man zum Beispiel den genauen Zeitpunkt bestimmen, an dem eine Frequenz eingesetzt hat, so bentigt man eine hohe Auflsung im Zeitbereich. Dies hat jedoch zur Folge, dass der Frequenzbereich unscharf wird. Um dieses Szenario zu erreichen muss die Fenstergre klein whlen.

  • Mchte man jedoch die Frequenz genau bestimmen, muss die Fenstergre gro gewhlt werden, damit man eine hohe Auflsung im Frequenzbereich erreicht. Hierbei wird die Zeitauflsung jedoch ungenau.</ul>

    Im Nachfolgenden sieht man diese Eigenschaft an einem Signal, indem die Frequnez zum Zeitpunkt t = 5 von 1 Hz auf 5 Hz springt:

Interaktive Simulation der Short-Time Fourier Analyse¶

Zuerst kann ein Signal ausgewhlt und angepasst werden oder auch ein eigenes Signal aus einer .wav Datei geladen werden.

signal = None

def set_signal(signal_chosen, duration_chosen, fs_chosen):
    global signal 
    
    # Die Lnge des Signals in Sekunden:
    duration = duration_chosen
    # Abtastrate des Signals in Hz
    fs = fs_chosen
    # Anzahl der "Abtastungen" innerhalb des gesamten Signals -> Lnge des Arrays fr das Signal
    N = int(duration* fs)
    # Erstellung des Zeitarrays fr die Dauer des Signals und der entsprechenden Abstastfrequenz
    time = np.arange(N) / fs
    # obiges Beispielsignal als "Frequency Step Signal"
    
    if signal_chosen is 0:
        f1 = 1
        f2 = 5
        t1 = time[:N//2]
        t2 = time[N//2:]
        x1 = 1.0 * np.sin(2 * np.pi * f1 * t1)
        x2 = 0.7 * np.sin(2 * np.pi * f2 * t2)
        signal = np.concatenate((x1, x2))
    elif signal_chosen is 1:
        # Chirp Signal
        signal = scipy.signal.chirp(time, 6, 10, 1)
    elif signal_chosen is 2:
        # Sinus
        signal = np.sin(2 * np.pi * time)
    else:
        signal = frequency_step_signal
    plt.plot(time, signal)
    plt.show()

interact(set_signal,
        signal_chosen=[("Cirp Signal", 1), ("Frequency Step Signal", 0), ("Sinus", 2)],
         duration_chosen=FloatSlider(min=1, max=100, step=0.1, continuous_update=False, value=10, description='Duration'),
         fs_chosen=fixed(fs));
